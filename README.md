## 📌 Project Overview

This is a fully custom-built single-page website I developed for my client **Risel Growth Agency**. The project involved designing and building a professional, modern, and fully responsive website from scratch using React and Vite — with smooth animations, an infinite brand logo slider, integrated contact form, and mobile-optimised layout.

---

## 🔗 Live Demo

👉 **[riselgrowthagency.com](https://riselgrowthagency.com)**

---

## 🎯 Client Brief

The client needed a website that:
- Looked premium and professional
- Showcased their services clearly
- Displayed client brand logos in an animated way
- Had a working contact form
- Was fully responsive on all devices
- Could be deployed and maintained easily

---

## ✦ What I Built

| Section | Details |
|:--------|:--------|
| 🏠 **Hero** | Animated headline, floating stat badges, orbital graphic |
| 📊 **Stats Bar** | Animated counters triggered on scroll |
| ⚙️ **Services** | 5 service cards with hover effects |
| 🏢 **About** | Dark section with animated reveal cards |
| 🏆 **Brand Slider** | Infinite dual-row logo marquee with fade edges |
| 💬 **Testimonials** | 6 client reviews with brand logos |
| 📬 **Contact Form** | EmailJS integrated, validated form |
| 📱 **Social Links** | Instagram and TikTok follow section |
| 🦶 **Footer** | Clean dark footer |

## 🚀 Running Locally

```bash
# Clone the repo
git clone https://github.com/amjadazizshah/Risel-Agency.git

# Install dependencies
cd Risel-Agency
npm install

# Start dev server
npm run dev

# Build for production
npm run build
```

## 📦 Deployment

Deployed on **Render** as a static site:

```
Build Command:     npm run build
Publish Directory: dist
Auto Deploy:       Enabled on every git push
```
